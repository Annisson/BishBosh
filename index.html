Bish-Bosh 2.0

This game will list all the numbers between 1 and your choosing from the input field below(loop). If the current number from the loop can be divided with either or both numbers you have entered for Bish and Bosh below, it will replace that number with either Bish, Bosh or Bish-Bosh. For example if you enter 10 in the loop-field, the program will run all the numbers between 1-10. If you chose 3 as Bish, and 4 as Bosh, it will print; 1, 2, Bish, Bosh, 5, Bish, 7, Bosh, Bish, 10.
